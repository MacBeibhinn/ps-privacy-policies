Permissions

The app requires certain device permissions to provide core launcher functionality, including organizing the home screen, displaying widgets, showing notification dots, and listing installed apps. These permissions are used solely for these core features and do not result in any data collection, transmission, or sharing.

The following permissions may be requested:

• Query Installed Apps (QUERY_ALL_PACKAGES): Required to display a list of installed apps. This data is only used to populate the app drawer(s) and any home page favorites/shortcuts, and to provide the user with a list of installed applications. This data is not used for any other purpose.

• Bind App Widgets (BIND_APPWIDGET): Required to allow adding and managing widgets on the home screen.

• Vibrate (VIBRATE): Required to allow the app to trigger device vibration, which is only used to provide haptic feedback when clicking on views, opening pages, etc.

• Receive Boot Completed (RECEIVE_BOOT_COMPLETED): Required to allow the app to run smoothly and make any important adjustments when the device has booted.

• Request Delete Packages (REQUEST_DELETE_PACKAGES): Required for requesting to delete or uninstall apps, applications will only be requested for deletion if the user selects the 'Uninstall' option within the app.

• Set Wallpaper (SET_WALLPAPER): Required for choosing and setting the device wallpaper directly from the app.

• BIND_NOTIFICATION_LISTENER_SERVICE & Notification Access: Required to show optional notification dots (or badges), indicating app notifications. This access is used only to check the presence and count of notifications associated with each app, and this information is not stored or transmitted.

All permissions are used exclusively on the device to enable the launcher’s features. No data accessed through these permissions is collected, stored externally, or shared with us or third parties.

Local Data Storage

The app stores certain preferences and configurations directly on the device to remember how the home screen is set up. This data includes:

• Chosen app shortcuts (or favorites) and their positions on the home screen.

• Selected widgets and their placement.

• Any custom settings applied within the launcher, such as grid size, themes, text or clock settings, icon packs, etc.

This data is stored locally on the device only and is never transmitted to external servers or shared with us or any third parties. Uninstalling the app will remove this locally stored data.
